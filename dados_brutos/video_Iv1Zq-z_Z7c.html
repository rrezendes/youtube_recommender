A parte II ainda abordando o assunto sobre conversão de objetos em query string no javascript.

A idea de ter gravado a versão 2 foi que muitos devs mandaram sugestões de várias outras formas de conversão de objeto em query string no javascript.

Inclusive formas que eu nem mesmo sabia. Então, resolvi gravar a versão 2 para deixar o vídeo ainda mais completo.

Na web, uma string de consulta faz parte de um URL que atribui valores aos parâmetros especificados. Para adicionar uma query string a uma URL você coloca o sinal de interrogação (?) após a URL e então adiciona uma série de par de parâmetros.

Exemplo: www.meusite.com?campo=valor&campo=valor.

Perceba que caso queira adicionar mais valores você adiciona o símbolo & e segue o mesmo padrão de campo e valor.

Um exemplo real seria: www.meusite.com?cidade=Salvador&idade=30.

Neste vídeo eu vou te ensinar como converter um objeto javascript nessa string de forma dinâmica e rápida. Vou te apresentar um passo-a-passo e também vou ter ensinar a forma ninja, que é como eu sempre gosto! 😁

Serão mais de 20 aulas onde você irá aprender os principais comandos de Python para machine learning e análise de dados. É importante que você conheça os recursos dessa linguagem de programação aplicados a um dos ramos de conhecimento que mais crescem no mundo: a inteligência artificial.
Machine Learning significa aprendizado de máquina, é uma técnica onde o computador aprende informações que não necessariamente foram escritas em código. Ou seja, seguindo uma metodologia específica de aprendizado, o computador consegue obter resultados e identificar padrões que os seres humanos não inseriram no código.
Python é uma linguagem de propósito geral, mas você não precisa ser um especialista em todas as esferas de Python para conseguir aplicar essa linguagem ao ramo do machine learning e da ciência de dados. Basta que você aprenda os comandos de programação mais importantes para essas áreas específicas e pronto, você estará apto a usar essa linguagem no dia a dia de trabalho.

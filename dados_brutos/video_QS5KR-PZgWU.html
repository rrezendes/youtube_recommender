Feature engineering is a key part of producing a machine learning model. In the words of Stanford professor and machine learning guru Andrew Ng, "Applied machine learning" is basically feature engineering.”

In this tutorial I will walk you through some basic feature transformations we can do on the Kaggle Titanic machine learning problem that should improve the model.

There are essentially endless things that you can do in feature engineering, and it is that feature engineering that sets the great models apart from the ones that are just OK.

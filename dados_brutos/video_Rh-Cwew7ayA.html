Have you ever wondered how self-driving cars are running on roads or how Netflix recommends the movies which you may like or how Amazon recommends you products or how Google search gives you such accurate results or how speech recognition in your smartphone works or how the world champion was beaten at the game of Go?

Machine learning is behind these innovations. In recent times, it has been proven that machine learning and deep learning approach to solving a problem gives far better accuracy than other approaches. This has led to a Tsunami in the area of Machine Learning.

This Webinar is the first and open session of Artificial Intelligence and Deep Learning Course by IIT Roorkee. It will be an introductory session about machine learning. If you like the session you can enroll and continue the course. Course certificate will be awarded by IIT Roorkee after course completion.

The agenda for this webinar is:

Introduction to Machine Learning
Machine Learning - Examples
Machine Learning - Applications
Hands-on Demo on Machine Learning
Q&A

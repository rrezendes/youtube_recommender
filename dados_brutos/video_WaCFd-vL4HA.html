In this video, I show you how you can build a deep learning model to detect melanoma with a very high accuracy. Melanoma is skin cancer which is very deadly but can be prevented if we can do early detection. Models like these might be useful to medical community.

I am using PyTorch with a "twist". The twist helps us to create highly customizable models quite fast!

I used the dataset from SSIM-ISIC which can be found here: https://www.kaggle.com/c/siim-isic-me...

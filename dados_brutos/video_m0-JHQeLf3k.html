- Discussion of the Freesound Audio Tagging 2019 competition: main ideas and winners’ approaches
- Description of the general approach to audio processing in DL: transition to the image classification problem and augmentation for the spectrograms
- Example of Kaggle API usage for automated models’ generation and blending

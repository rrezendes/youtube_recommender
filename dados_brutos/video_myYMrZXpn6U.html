KAGGLE TUTORIAL! HANDLE LARGE IMAGE DATASETS LIKE A DATA SCIENTIST

In this kaggle tutorial you'll be learning how to handle large image datasets in kaggle like a data scientist! We'll be using jupyter notebooks, when you create your kaggle kernel you'll also going to be using a jupyter notebook.

If you never heard of jupyter notebook you can checkout the site docs here
https://jupyter.org/
jupyter notebook are a great tool to make shareable and replicable code, a jupyter notebook was used on the LIGO research, how cool is that? They grew in popularity as well as anaconda ( python library manager ) , one thing great about jupyter notebook is that you can publish your jupyter notebook on github and your jupyter notebook will be rendered in the page, pretty cool :D

But let's go to the facts. Handling large image datasets in kaggle competitions can be a pain sometimes, even if you follow a kaggle tutorial ( usually a jupyter notebook btw ).

But don't you worry! This is the time where we get our hands on working with this nasty little things in those awesome kaggle competitions!

If you don't know kaggle at all, you can check out their website here
https://www.kaggle.com/
It's a great place to get started, you can use the kaggle kernels and kaggle tutorials as guide
You can check out the kernels ( kaggle tutorial ) here
https://www.kaggle.com/kernels
If you want to create your own Kernel you'll problably want to get familiar with jupyter notebook :D.

You could also go to a competition that you've liked and search for a specific kaggle tutorial on them
https://www.kaggle.com/competitions
kaggle also has a discussion part, you can find some kaggle tutorial in this section as well
https://www.kaggle.com/discussion

In this kaggle discussion section you can find kaggle tutorial like this one
https://www.kaggle.com/c/crowdflower-...

You can also participate into discussions to get medals, this will get you ranked on the kaggle website. You can get ranked by posting new kernels ( like a kaggle tutorial ), or you could help some one on the discussion forum. Or you could participate into competions.
Kaggle has 5 levels in all of the above, kaggle competition, kaggle discussion and kaggle kernel ( kaggle tutorial )

#1 Novice
#2 Contributor
#3 Expert
#4 Master
#5 Grandmaster

Also, when you create your own kaggle tutorial and see the feedback of the community this will help you grow as a data scientist into the community, you can check out the new state of the art models that are used in this kaggle tutorials.


The code for the kaggle tutorial is here, if you've liked, consider leaving a vote!

https://www.kaggle.com/jhonatansilva3...

PyTorch dataloader tutorial

https://pytorch.org/tutorials/beginne...

In this tutorial we're going to be going through

1. What libraries we use in DataScience projects
2. Creating a class in Python for our Dataset
3. Iterating through the dataset
4. Preprocessing
5. Putting all together

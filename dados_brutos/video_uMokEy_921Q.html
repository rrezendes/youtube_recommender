Join in to hear from Machine Learning Experts working across various teams at Google.

This bootcamp aims to provide mobile developers an introduction to machine learning and equip them with necessary knowledge to integrate ML-powered features to their mobile apps. It focuses on on-device ML technologies like TensorFlow Lite, ML Kit and AutoML Edge.

Website: http://bit.ly/mlb-mobile-devs
Speakers: http://bit.ly/mlb-speakers
Submit your feedback here: http://bit.ly/mlb-challenge-submit

Agenda:

Introduction to machine learning for mobile developers (by Khanh LeViet, Developer Advocate, TensorFlow) : 10 mins

Train a flower classifier with TF Lite Model Maker and integrate it to an Android app (by Wei Wei, Developer Advocate, TensorFlow) : 30 mins

Leverage TF Lite pre-trained models (by Khanh LeViet, Developer Advocate, TensorFlow) : 10 mins

Introduction to ML Kit and ML APIs on Android (by Hoi Lam, Developer Relations Lead, Android ML) : 40 mins

Introduction to Cloud ML APIs and AutoML Vision Edge (by Kaz Sato, Developer Advocate, Google) : 30 mins

This KNIME tutorial covers using the random forest model to make predictions for the Kaggle Titanic: Machine Learning from disaster problem.

The random forest is an easy to use, adaptable, and robust to overfitting machine learning model. It is also highly capable for classification problems and thus a great choice for this Kaggle.

The random forest is a collection of decision trees that are bootstrapped through random selection with replacement. I recommend also checking out these videos on the decision tree model from KNIME data scientist Rosaria Silipo.
